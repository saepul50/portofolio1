Tentang

Hai, saya Yaskur Habi Abdillah seorang mahasiswa baru dan aktif dari prodi Teknik Informatika Politeknik Elektronika Negeri Surabaya atau lebih dikenal dengan PENS. Saya mulai mengenal dunia pemrograman sejak SMK karena ketertarikan saya pada bidang Informatika dengan memilih jurusan Rekayasa Perangkat Lunak (RPL). Dari situlah saya mulai mempelajari dasar-dasar pengembangan perangkat lunak.

Saat ini, saya terbiasa di bidang front-end maupun back-end. Di front-end, saya memiliki pengalaman dalam membangun tampilan website yang responsif dan interaktif menggunakan HTML, CSS, JavaScript, serta beberapa framework. Di back-end, saya terbiasa mengelola logika aplikasi, database, serta API menggunakan PHP dan MySQL. Dengan hal ini saya bisa mengembangkan aplikasi dari sisi UI/UX hingga dari sisi back-end.
